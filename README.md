stockcmd
===================

本專案由 Python 撰寫，提供一個快速在 terminal 上取得台股報價的功能！

---------
**格式：**  

    stockcmd.py [Options] [Stock_No]

**參數：**  

    -a: 列出所有資訊 (國外股市、台灣上市櫃指數、個股報價)
    -i: 增加上市&上櫃指數的報價
    -w: 顯示國外股市行情
    -q: 只顯示存在 List 中的股票清單 (不需輸入股號)
    -c: 顯示彩色畫面
    -h: 顯示說明頁面
